# iKunPong

篮球皮肤之Pong(简单版)

LEFT&RIGHT键控制板子

## 游玩说明

1. `build`文件夹和`ikunpong.spec`是`pyinstaller`自动生成的构建文件，不用管它.

2. **请把`img`文件夹和EXE文件放在同一目录，否则无法游玩**

**正确目录**:
```
iKunPong
   |--[build]
   |--[img]
   |    |--(ball.png)
   |    |--(paddle.png)
   |--(ikunpong.exe)
   |--(ikunpong.pyw)
   |--(ikunpong.spec)
```

> 学习pygame时的产物，bug肯定有的，也就只是拿来学习pygame用的
