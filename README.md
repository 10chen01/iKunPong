# iKunPong

篮球皮肤之Pong(简单版)

LEFT&RIGHT键控制板子

## 游玩说明

1. `build`和`.spec`文件不用管，构建的文件

2. 运行命令：
```shell
source bin/activate
pip3 install -r requirements.txt
python3 ikunpong.pyw
```

**正确目录**:
```
iKunPong
    |--[build]
    |--[img]
    |    |--(ball.png)
    |    |--(paddle.png)
    |--(ikunpong.pyw)
    |--(ikunpong.spec)
    |--[...]
```

> 学习pygame时的产物，bug肯定有的，也就只是拿来学习pygame用的
